❓ FAQs

HEX is hexadecimal color format like #FF5733. First 2 chars = Red, next 2 = Green, last 2 = Blue. Each from 00 (0) to FF (255).

RGB: Red, Green, Blue (0-255)
HSL: Hue (0-360°), Saturation (0-100%), Lightness (0-100%)
HSL is more intuitive for designers.

CMYK (Cyan, Magenta, Yellow, Key/Black) is used for print design. RGB for screens, CMYK for printers.
